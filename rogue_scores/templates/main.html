Install Rogue

On GNU/Linux

wget http://rogue.rogueforge.net/files/rogue5.4/rogue5.4.4-src.tar.gz
tar xzf rogue5.4.4-src.tar.gz && cd rogue5.4.4-src
./configure --disable-debug --disable-dependency-tracking
[sudo] make install

On OS X

brew install rogue
mkdir -p /usr/local/var/rogue
