Install Rogue

On GNU/Linux

wget http://rogue.rogueforge.net/files/rogue5.4/rogue5.4.4-src.tar.gz
wget https://raw.githubusercontent.com/bfontaine/rogue_scores/master/utils/rogue-scorefile.patch
tar xzf rogue5.4.4-src.tar.gz && cd rogue5.4.4
./configure --disable-debug --disable-dependency-tracking
patch < rogue-scorefile.patch
mkdir -p /var/rogue
[sudo] make install

On OS X

brew install rogue
